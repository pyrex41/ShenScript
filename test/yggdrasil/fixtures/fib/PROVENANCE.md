# Provenance

These files are a Yggdrasil stage-1 output, copied verbatim from
`Yggdrasil/out-demo/` (the fib demo shaken against ShenOSKernel-41.2,
eval-stripped: 102 kernel forms, `needs-eval=false`).

To regenerate, follow Yggdrasil's DEMO.md: run `yggdrasil.shake` on
`fib.shen` with a certified Shen (e.g. shen-cl) and copy `kernel.kl`,
`fib.kl` and `yggdrasil.manifest.txt` here.
